The Financial Analyst Assistant Portal is a web-based application designed to assist financial analysts in extracting insights from annual reports using Natural Language Processing (NLP) and Deep Learning (DL) techniques. This portal enables users to pose questions related to financial documents and receive accurate, context-aware answers generated from the reports.

**Question Answering System**: Users can input questions about financial metrics, performance, and other relevant details, and the system retrieves precise information from the annual reports.
